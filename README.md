# ToDo-List

ToDo list is a simple web application to save Your daily tasks in order not to miss anything.

### Features
* Add new task
* Mark task as completed / uncompleted
* Delete single task
* Access THE COMPLETED/ COMPLETED/ALL TASKS


### Technologies used:
* HTML
* CSS
* Vanilla JavaScript
